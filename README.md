This is a [React 19](https://react.dev) project initialized with [Vite](https://vitejs.dev/).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `src/pages/Home.tsx`. The page auto-updates as you edit the file.

## Deploy

You can deploy this project using any static site hosting service (Vercel, Netlify, Cloudflare Pages, etc.) by running the build command:

```bash
npm run build
```

This will produce a `dist` folder that you can deploy.
